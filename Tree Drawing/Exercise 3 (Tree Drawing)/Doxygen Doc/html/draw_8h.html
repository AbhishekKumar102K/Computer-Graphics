FUNCTION TO ILLUMINATE PIXELS BY 8-WAY SYMMETRY.

If pixel (x,y) is illuminated, then (x,-y),(-x,y),(-x,-y),(y,x),(y,-x),(-y,x) and (-y,-x) are also illuminated, since a circle is symmetric about the lines y = x and y = -x

Parameters
xX coordinate of the pixel to be illuminated
yY coordinate of the pixel to be illuminated
xoffX coordinate of the centre of the circle
yoffY coordinate of the centre of the circle
